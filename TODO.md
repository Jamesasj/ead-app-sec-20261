
[ ] OWASP A1: Injection 
    - Falta de validação de entrada em endpoints, permitindo injeção de SQL
[ ] OWASP A2: Broken Authentication 
    - Falta de autenticação e autorização adequada, permitindo acesso não autorizado a funcionalidades do sistema
[ ] OWASP A3: Sensitive Data Exposure 
    - Senha do usuarios e salva sem critografia
[ ] OWASP A4: XML External Entities (XXE) 
    - Falta de validação de entrada em endpoints que processam XML, permitindo ataques de XXE
[ ] OWASP A5: Broken Access Control 
    - Falta de controle de acesso adequado, permitindo acesso não autorizado a funcionalidades do sistema
[ ] OWASP A6: Security Misconfiguration 
    - Configurações de segurança inadequadas, como falta de configuração de CORS, CSRF, etc.
[ ] OWASP A7: Cross-Site Scripting (XSS) 
    - Falta de validação de entrada em endpoints que processam dados do usuário, permitindo ataques de XSS
[ ] OWASP A8: Insecure Deserialization 
    - Falta de validação de entrada em endpoints que processam objetos serializados, permitindo ataques de deserialização insegura
[ ] OWASP A9: Using Components with Known Vulnerabilities 
    - Uso de bibliotecas e componentes com vulnerabilidades conhecidas
[ ] OWASP A10: Insufficient Logging & Monitoring 
    - Falta de registro e monitoramento adequado, dificultando a detecção e resposta a incidentes de segurança
